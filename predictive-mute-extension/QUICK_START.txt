⚡ HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Monitors keyboard input on meeting tabs
2. When Enter pressed, analyzes text word-by-word
3. For each word, checks next N words (lookahead)
4. If sensitive keyword found → triggers mute
5. Shows visual alert + logs event

🎓 KEY CONCEPTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Lookahead Window:
  Number of words to check ahead. Higher = more cautious but
  may trigger earlier than needed.

  Lookahead=2: Checks current word + next 2 words
  Lookahead=5: Checks current word + next 5 words

Word Normalization:
  Removes punctuation and makes lowercase for matching
  Example: "Password!" → "password"

Partial Matching:
  "pass" will match "password", "passphrase", etc.
